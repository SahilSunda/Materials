1. Front-end Developer:

Your Opportunity:

You will primarily be responsible for designing, developing, and testing UI for mobile and web applications. You will possess strong skills in object-oriented software design and build reusable code and libraries for future use. You are excited about developing new features, maintaining existing code, fixing bugs, and contributing to overall system design.

Your Impact:

To design, architect, develop and maintain the Front-End for AI products by working closely with the Engineering Team. To ensure successful consumption of Saarthi.ai technology by APIs, SDKs, or Applications, thereby enabling productization and monetization of the AI solutions.

What You’ll Do:

  • Developing Web Apps and maintaining them
  • Build reusable code and libraries for future use
  • Accurately translate user and business needs into functional frontend code
  • Plan and implement new designs, Optimize existing user interface designs
  • Create wireframes, interactive prototypes to communicate.
  • Should collaborate and work with and development team to define and implement innovative web and mobile solutions, visuals, and experience.
  • Follow mobile and web platform guidelines.
  • Design UI's as per requirements.
  • Create prototypes for new product ideas.
  • Develop the front end of an application using JavaScript framework for web, android, and IOS.

What You Bring:

  • Proven experience as a Front-End Developer or similar role
  • Experience developing desktop and mobile applications
  • Familiarity with common stacks
  • Knowledge of multiple front-end languages and libraries (e.g., HTML/ CSS, JavaScript, XML, React, jQuery, TypeScript)
  • Knowledge of Front-End languages (JavaScript), JavaScript Frameworks (React, Angular, Node.js)
  • Excellent communication and teamwork skills
  • Degree in Computer Science, Statistics or relevant field

Would-Be Nice If You Also Have:

  • Ability to write quality unit tests
  • He or she should have a complete understanding of automated processes for building testing, document, and deploying it at scale
  • An awareness of security concerns is important, as each layer has its vulnerabilities
  • Knowledge of Algorithms and data structures is also an essential need for professional Full-Stack developers
  • Knowledge of PhotoShop, Figma, or other designing and prototyping tools.
